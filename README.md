English
File Renamer Tool
This Python script allows you to rename files in a specified directory, including its subdirectories, by replacing specific text in the filenames. You can customize the text to search for and the replacement text. The tool logs all changes in a log file for easy tracking.

Features:

Recursive renaming of files in a directory.
Customizable search and replace text.
Logs all actions in file_rename.log.
User-friendly console interface.
How to Use:

Run the script (or .exe) and provide the directory path.
Enter the text to search for and the replacement text.
The tool will rename the files and display the changes in the console and log file.
Deutsch
Datei-Umbenennungs-Tool
Dieses Python-Skript ermöglicht das Umbenennen von Dateien in einem angegebenen Verzeichnis, einschließlich Unterordnern, indem bestimmter Text in Dateinamen ersetzt wird. Der zu suchende Text und der Ersatztext können angepasst werden. Das Tool protokolliert alle Änderungen in einer Logdatei, um die Nachverfolgung zu erleichtern.

Funktionen:

Rekursives Umbenennen von Dateien in einem Verzeichnis.
Anpassbarer Such- und Ersetzungstext.
Protokolliert alle Aktionen in file_rename.log.
Benutzerfreundliche Konsolenoberfläche.
Anwendung:

Führen Sie das Skript (oder die .exe) aus und geben Sie den Verzeichnispfad ein.
Geben Sie den zu suchenden Text und den Ersatztext ein.
Das Tool benennt die Dateien um und zeigt die Änderungen in der Konsole und Logdatei an.
